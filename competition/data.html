Clustering description

In the Clustering stage, you have given a collection of news documents from the Telegram Data Clustering Contest and a set of pairs from these documents. Your task is, for each given pair, to predict whether two documents describe the same event. We provide the labeled training dataset and the unlabeled test dataset.

Clustering columns
  • first_url - URL of the first document
  • second_url - URL of the second document
  • quality - "OK" if two documents describe the same event, "BAD" otherwise
  • dataset - exists only in test dataset to distinguish different dates
